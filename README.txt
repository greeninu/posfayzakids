=== POS Fayzakids — Android App ===

CARA BUILD APK:
──────────────────────────────────────────────
1. Install Android Studio (gratis): https://developer.android.com/studio

2. Buka folder "pos_android" ini di Android Studio
   (File → Open → pilih folder pos_android)

3. Tunggu Gradle sync selesai (butuh internet, ~5 menit pertama kali)

4. Build APK:
   - Menu: Build → Build Bundle(s)/APK(s) → Build APK(s)
   - Tunggu selesai, klik "locate" untuk buka folder APK

5. Install APK ke HP Android:
   - Aktifkan "Install dari sumber tidak dikenal" di Settings HP
   - Copy APK ke HP via USB / kirim WhatsApp ke diri sendiri
   - Tap file APK untuk install

CARA PAKAI DI APK:
──────────────────────────────────────────────
1. Pastikan printer Blueprint Lite 58 sudah di-PAIR di Settings Bluetooth HP
   (Settings → Bluetooth → Scan → pilih printer → masukkan PIN 0000)

2. Buka app POS Fayzakids

3. Masuk ke menu Setting → Printer

4. Tap "Hubungkan Printer" → muncul dialog daftar device Bluetooth yang sudah di-pair

5. Pilih printer Blueprint Lite 58 → otomatis terhubung

6. Setelah itu setiap transaksi akan langsung tercetak otomatis!

CATATAN:
──────────────────────────────────────────────
- App memerlukan koneksi internet (memuat pos.fayzakids.com)
- Bluetooth Classic (SPP) — printer yang pakai PIN 0000 — DIDUKUNG penuh
- Minimal Android 6.0 (hampir semua HP Android saat ini)
